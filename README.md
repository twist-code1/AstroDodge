# AstroDodge
3D dynamic Shooter built with p5.js and WEBGL featuring dynamic difficulty and custom .obj models

## 🔗 Link
[Play here](https://twist-code1.github.io/AstroDodge/)

## 🛠 Tech Stack
* **Language:** JavaScript (ES5+)
* **Engine:** p5.js / WebGL
* **Assets:** Custom `.obj` models + Web Audio API
* **State:** LocalStorage for high scores

## ✨ Key Features
* **Dynamic Scaling:** Speed increases based on travel distance.
* **Barrel Roll:** `Up` key trigger. Dodges obstacles + ammo reload mechanic.
* **Game Juice:** Implemented screen shake on collisions/firing.
* **Optimization:** Low-poly models and particle pooling for stable FPS.

## 🕹 Controls
* `WASD / Arrows` — Movement
* `Up` — Barrel Roll (Dodge/Reload)
* `Down` — Brake
* `Space` — Shoot

## ⚠️ Compatibility
* **Desktop only.** No mobile/touch support yet.
* Requires a local server (CORS) for local dev.
